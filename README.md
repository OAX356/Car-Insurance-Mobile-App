# 🚗 Mobile Car Insurance Application

A cross-platform mobile solution designed to streamline vehicle insurance profile management, claims, and administrative tracking.

## 📌 Features & System Capabilities
- **Role-Based Access Control (RBAC):** Customized user flows and permissions for Clients, Insurance Agents, and System Administrators.
- **Real-Time Data Sync:** Live vehicle status tracking and claims notifications powered by Firebase Cloud Messaging.
- **Cross-Platform UI:** Native performance across Android and iOS built using Flutter/Dart.

## 🛠️ Tech Stack & Backend Services
- **Frontend / Mobile Framework:** Flutter, Dart
- **Backend Infrastructure:** Firebase Authentication, Cloud Firestore, Firebase Storage
- **State Management:** Provider Pattern

## 📱 Application Flow Overview
- **Client Portal:** Submit claims, upload vehicle photos, view policy details in real time.
- **Agent Portal:** Review incoming claims, update claim processing statuses, push updates.
- **Admin Console:** Manage system roles, audit claims, and monitor overall analytics.
